# Pythonコードを書くときのスキルを定義します

## 前提条件
このSKILLはPythonプログラムを書くように指定された時に用います
コードを書くときは基本はLinuxに適したコードを書きます。ただし、なるべく、WindowsでもmacOSでも動作可能なプログラムを心がけてください。
sandoboxで動かすときはこのSKILLは使いません

## プログラムを書くとき
関数にはDocstringsを記載してください。ごく簡単なプログラムについては不要ですが、何かしらの分岐、繰り返し、変換などがある場合はDocstirngsを記載してください。
長いプログラムの場合は、全体の流れを説明した上で、粒度を少しずつ細かくして説明をしてください。
まず最初に全体像を説明することが大事です。

## 解説を求められた時
プログラムで何を行なっているか、詳細に解説してください。
長いプログラムの場合は、全体の流れを説明した上で、粒度を少しずつ細かくして説明をしてください。
まず最初に全体像を説明することが大事です。
特に、デコレータ、WokerPoolなどの非同期については丁寧に解説してください。

## バージョンの明確化
Pythonのどのバージョンで書いているかを明確にしてください。
Python2は使わない。必ず、Python3を使ってください。
